# Screenshots

README が参照する画像置き場。下の名前で撮って置けば README に自動で表示されます。
1280×800 か 1440×900 くらいで OS のスクリーンショット機能 (mac: ⌘⇧4) で撮るのが推奨。

| File | 撮るもの |
|------|---------|
| `hero.png` | アプリ全体。Live Preview モードで sample.md などを開いた状態。アプリの第一印象になるので一番きれいな状態で |
| `diff-sbs.png` | Diff モード → Side-by-Side サブモードで、左右に違いがハイライトされた状態。base picker に `HEAD~3` あたりが見えてると◎ |
| `history-view.png` | 「この版を見る」で履歴ビューに入った状態。上の `Viewing read-only at: ...` バナーとレンダリングされた本文が両方写るように |
| `history-diff.png` | 履歴中に Diff モードに切り替えて、「pinned 版 vs HEAD」の chip が picker 横に出てる状態 |
| `outline.png` | アウトラインサイドバーが右側に出てる状態。見出しが多めのファイルを使うと差別化が出る |
| `tui.png` | （任意）TUI を端末で開いた状態 |
